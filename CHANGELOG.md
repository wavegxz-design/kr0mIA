# Changelog — Kr0m

## [1.0.0] — 2025

### Added
- Multi-model support: OpenAI, DeepSeek, Gemini, Mistral, Anthropic, Groq, Together
- Vault AES-256-GCM con PBKDF2 600k iteraciones
- Tor routing via torsocks
- Modo multi-modelo paralelo con timer por modelo
- Syntax highlighting con bat
- Prompts especializados: CTF, RE, PWN, OSINT, Recon, Dev, Bash, Study
- Modo interactivo con comandos internos
- Stealth mode (sin logs ni historial)
- Auto-clear history
- Panel de settings TUI
- Sesiones con historial encriptable
- Timer de respuesta y contador de tokens
- Guardar respuestas a archivo Markdown
- Instalación global via install.sh
- Vault con 4 métodos: password, password+keyfile, UID, password+TOTP
